# Disease Prediction from Medical Data

## Project Overview

This project predicts the likelihood of heart disease using patient medical information and machine learning techniques.

## Objective

To develop a classification model that assists in early disease detection based on medical attributes.

## Dataset

Heart Disease Dataset

Features include:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Maximum Heart Rate
* Exercise Induced Angina

Target Variable:

* Presence of Heart Disease

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Machine Learning Algorithm

* Logistic Regression

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score

1. Install dependencies:

pip install -r requirements.txt

python disease_prediction.py
